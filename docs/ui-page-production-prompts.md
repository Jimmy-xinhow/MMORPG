# UI Page Production Prompts

> Updated: 2026-05-27
> Target: Godot 4 mobile portrait client, 432x768 per page, 864x1536 2x2 contact sheet.

## Style Lock

All future page batches must match the current production UI style:

- Dark navy/black fantasy MMORPG client chrome.
- Ornate gold bevel frames and compact dense panels.
- Fixed top character HUD with portrait, HP/EXP, currencies, task/activity buttons.
- Fixed bottom seven-icon navigation bar with active blue-gold glow.
- Chibi/anime original fantasy characters and monsters.
- Blue-gold buttons, small red notification badges, green progress bars.
- No parchment-dominant western form style unless it is only a small inset panel.
- No real-money, cash-out, withdrawal, admin, settlement, or operator finance copy in player UI.
